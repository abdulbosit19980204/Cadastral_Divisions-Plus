Cadastral Divisions Plus

Poligonlarni kesib, n ta teng bo‘lak yoki maqsad yuzasi asosida yangi qatlam hosil qilish uchun kengaytirilgan QGIS Processing plagini.

Yangi qo‘shilgan + lar

  • Residual (qoldiq) boshqaruvi: oxirgi bo‘lak yetarli bo‘lmasa, qoldiqni birinchi/oxirgi/tanlangan bo‘lakka qo‘shish yoki alohida qoldirish.
  • Ko‘p til: italiyan, ispan, inglizga qo‘shimcha ravishda ruscha va o‘zbekcha UI matnlari.
  • Rebrending: modul fayllar va Processing guruhi `Cadastral Divisions Plus` nomi bilan ro‘yxatdan o‘tadi — original plagin bilan to‘qnashmaydi.
  • Yangi qatlam prefiksi: Natija fayllari `FractPlus_YYYY-MM-DD HH:MM:SS` nomi bilan yaratiladi.
  • Stabillik: mavjud atributlar saqlanib, `sub`, `part`, `area` maydonlari avtomatik qo‘shiladi.

O‘rnatish

  1. Repo papkasini ZIP qiling yoki `Cadastral_Divisions-main` katalogini to‘g‘ridan-to‘g‘ri QGIS3/profiles/default/python/plugins/ ichiga nusxalash.
  2. QGIS’ni qayta ishga tushiring, Plugins > Manage and Install Plugins ichidan “Cadastral Divisions Plus” ni yoqing.
  3. Agar Plugin Reloader’dan foydalanayotgan bo‘lsangiz, toolbar ichidagi drop-down ro‘yxatdan shu nomni tanlang.

Ishlatish bo‘yicha qo‘llanma

Processing Toolbox > Cadastral Plus > Cadastral Divisions Plus algoritmini ishga tushiring.

ParametrTavsif
Input Poly layerBo‘linadigan poligon(lar).
Input Line layerBo‘luvchi chiziq; agar qisqa bo‘lsa, algoritm uni cho‘zadi va markazlashtiradi.
Split with divider manually positionedTrue bo‘lsa, chiziqqa hech qanday avtomatik tuzatish qo‘llanmaydi.
Denominator fraction or 1/n partsn ta teng bo‘lak yoki kasr maxraji.
Splitting into equal partsn > 2 bo‘lganda ketma-ket kesish rejimi.
Surface area to be obtainedMaqsad yuzasi (kvadrat birliklarda). Berilsa, ulush o‘rniga shu miqdor kesiladi.
Invert partsChiziqqa nisbatan yuqori/pastki yoki chap/o‘ng qismlarni almashtiradi.
Residual handling modeQoldiqni oxirgi, birinchi, tanlangan bo‘lakka qo‘shish yoki alohida qoldirish.
Residual target part“Manual” rejimda qoldiq biriktiriladigan bo‘lak raqami (1 dan boshlab).

Natijada yaratiladigan qatlamga original atributlar bilan birga quyidagilar qo‘shiladi:

  • sub ...: bo‘lak identifikatori (masalan, sub 1, sub qoldiq).
  • ..._part: bo‘lakning umumiy yuzaga foiz nisbati.
  • ..._area: real yuzasi.
